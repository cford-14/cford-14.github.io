Hangman

I wrote this program in Python to help my daughter with her spelling homework. It takes a list of spelling words and an allows the user to play Hangman with each word. The words appear in random order and only appear one time each.

My hangman program currently only runs in the terminal. It would need the following functionality:

  • User-friendly interface
  • Mechanism for inputting new spelling word lists
  • Graphic display of a hangman that changed with inccoret answers
  • Graphic display of blanks for the active word
  • Possibly a way to collect and display number of wins

Shopping List

This application would allow users to build several shared shopping lists so that any family member could pick up groceries or household supplies. Each list item could:

  • Contain a user-added photo so that the shopper could view all of the details of each item. This is particularly useful when shopping for favorite brands and toiletries
  • Items could appear on multiple lists. Sometimes an item can be purchased at the drug store or the grocery store.
  • Items could be checked off and deleted. An item that appears on more than one list will be deleted from all lists simultaneously.

This app could also email or text shopping lists at set times. It could also be adapted for Christmas and birthday wishlists.
